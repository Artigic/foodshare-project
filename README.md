Food Waste Sharing Web App

# 🍱 FoodShare - Food Waste Management System

## 🌍 SDG Goal
Responsible Consumption and Production

## 📌 Problem
Food waste is increasing while many people face food shortages.

## 💡 Solution
A web platform where users can share surplus food with others.

## 🚀 Features
- Add food listings
- View available food
- Claim food
- Simple dashboard

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## ▶️ How to Run
1. Run backend:
   node server.js
2. Open index.html in browser
